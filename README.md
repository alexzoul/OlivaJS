# Typescript, Webpack 2, Karma, Sass

### Requirements
* UNIX O.S. like Mac OSX or Linux.
* NodeJs v7.0.


### To watch this project in action you need:
  1. npm install -g yarn
  2. yarn install
  3. npm run build
  4. cd dist
  6. open index.html on your browser


### To start developing on this project you need:
  1. npm install -g yarn
  2. yarn install
  3. npm start
